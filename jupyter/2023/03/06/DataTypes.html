    public String BinaryConvert(int a)

        int temp = 0;

            //out+=String.valueOf(temp); 
            out = String.valueOf(temp)+out;
        }
        //out = reverseString(out);
        System.out.println(out);
        return out;
        //return Integer.parseInt(out, 2);   
    }

    private String reverseString(String out)
    {
    	char[] ch = out.toCharArray();
    	int x = out.length()-1;
    	String s="";
    	for(;x>=0;x--) {
    		s+=ch[x];
    	}
    	return s;
    }

Notes on Data Types

  • Object is the reference type.
  • int by reference type, maybe creating a class object, value is known by the class.
  • Inside of the structure, there are pointers to memory location inside of the class
  • The memory location called when creating a class is called the heap and use of pointers like in "C" to point to the at location.
  • int vs Integer:
    • int: stores 32 bit signed two complement integer
    • Integer: class that wraps a primitive data type in an object, Helps to convert int to object and to convert an object to int
  • Auto Boxing:
    • convert int to object and to convert an object to int
  • A string is a class, char is a single character
  • values of the array get stored in the heap.
  • Hashcode gives you the memory location of the Integer wrapper class.
public class Conversions{
    
    private int[] arr = new int[3];
    private double[] arr1 = new double[2];
    private boolean[] b = new boolean[2];
    private char ch;


    public Conversions(byte a1,short a2,int a3, float d1,double d2,
                boolean b1,boolean b2,char c1)
    {
        arr[0]=a1;
        arr[1]=a2;
        arr[2]=a3;
        arr1[0]=d1;
        arr1[1]=d2;

        Integer n1=Integer.valueOf(a1);

        "\n Short val:"+n2+
        "\n Int val:"+n3+
        "\n Float val:"+dd1+
        "\n double val:"+dd2+
        "\n boolean val1:"+bb1+

Key Concepts

  • Methods and control structure: Help to organize and control the flow of the code.
    • Method: is a block of code that takes in a parameters to produce a desired result
    • Control Structures: Statements that determine the order in which other statements in a program are executed. They are catogorized into 3 main categories: conditional statements, looping statements, branching statements
  • Diverse Arrays and Matrixes are examples of control structure because they use methods like reverse or arraySum which uses control structure statements like for loop and if statements
  • They fit data types too because of usage of reference type data types: arrays.
  • IntbyReference is important because this class uses pass by reference
